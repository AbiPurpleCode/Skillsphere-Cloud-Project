SkillSphere Cloud Architecture

1. Users access the web application through the public IP address of the server.
2. The application is hosted on an Amazon EC2 (Elastic Compute Cloud) instance running Ubuntu.
3. Docker is installed on the EC2 instance to manage containers.
4. An Nginx container runs inside Docker and serves the web application.
5. The application connects to Amazon RDS (Relational Database Service) which stores user data and course information.
6. Amazon S3 (Simple Storage Service) is used to store course materials and static files.

This architecture allows scalability, reliability, and easier infrastructure management compared to traditional on-premise infrastructure.
