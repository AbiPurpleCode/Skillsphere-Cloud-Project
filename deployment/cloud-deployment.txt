Cloud Deployment Steps for SkillSphere

1. Created an EC2 instance on AWS (Ubuntu Server).
2. Connected to the instance using SSH through the AWS console.
3. Installed Docker on the EC2 instance.
4. Pulled and ran an Nginx container using Docker.
5. Verified the web server by accessing the public IP in a browser.
6. Created an Amazon RDS MySQL database for storing application data.
7. Created an Amazon S3 bucket to store course files and static content.
8. Verified that all services were running successfully.
